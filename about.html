About

This quote generator site serves to make shareable images of your favorite quotes. I made it since I personally have a lot of quotes I oftentimes want to make cool images to accompany for the sake of sharing them, but don't want to take the time to manually edit them in a program. Using it is easy! Input the main text, original author, and image you want to use, format the text how you want, and save!
